## Run Locally

1. Clone the project

```bash
  https://github.com/aliseyedi01/Blog-App.git
```

2. Go to the project directory

```bash
  cd Blog-App
```

3. Install dependencies

```bash
  npm install
```

4. Start the server

```bash
  npm run dev
```
